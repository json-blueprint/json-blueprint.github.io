Expressive

JSON Blueprint patterns closely resemble matching JSON documents. 

Video = {
  id: String,
  dimension: "2d" | "3d",
  definition: "hd" | "sd",
  (hasThumbnail: Boolean)?
}

Simple

Just a handful of constructs that can be combined in powerful ways. 

Id = Int(min=0) | String
Resource = {
  id: $Id,
  (references: [$Id*]) |
  (embedded: [$Resource*])
}

Strict

Strict by default, allowing you to explicitly opt-out when you need it. 

ProbablyPerson = {
  firstName: String | null,
  lastName: String | null,
  // any other properties
  (String: Any)*
}
